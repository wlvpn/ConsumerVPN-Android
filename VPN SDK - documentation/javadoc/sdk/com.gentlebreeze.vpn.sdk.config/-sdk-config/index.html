SdkConfig

data class SdkConfig : VpnApiConfiguration, IpGeoConfiguration

Contains information needed to initialize the SDK in your application. accountName, apiKey, and authSuffix are required and will be provided. All other parameters are optional and, in most cases, will not be needed to modify.

Parameters

accountName - String value containing your account name

apiKey - String value containing your API key

authSuffix - String value containing your auth suffix

client - String value designating the client (e.g. Android)

apiHost - String value designating the API hostname

ipGeoUrl - String value designating the IP Geo API hostname

apiLoginEndpoint - String value endpoint for the API login

apiTokenRefreshEndpoint - String value endpoint for the API token refresh

apiProtocolListEndpoint - String value endpoint for the API protocol list

apiServerListEndpoint - String value endpoint for the API server list

logTag - String value for sdk logtag
