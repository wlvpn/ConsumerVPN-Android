connectToNearestRestrictedByCountry

abstract fun connectToNearestRestrictedByCountry(pop: VpnPop, notification: VpnNotification, vpnRevokedNotification: VpnNotification, configuration: VpnConnectionConfiguration): ICallback<Boolean>

Attempt to connect to a server from a given VpnPop if any available, after performing a load balance considering maintenance, server capacity and distance one will be taken as the best suited for the connection. If there is no candidate servers available in the specified VpnPop, the nearest one from other VpnPop will be used, expanding the tolerance distance, by checking the nearest ones, but never outside the specified VpnPop's country.

Parameters

pop - pop to attempt to connect to

notification - vpn notification

vpnRevokedNotification - notification shown if the VPN permission is revoked

configuration - Container object containing server, notification configuration, and credentials

Return
callback to establish the connection
